# Images Directory

This directory contains generated images for the AI Language Tutor.

## Directory Structure

### `images/grammar/` ✅ Active

Contains grammar pattern illustrations generated by `resources/generate_grammar_illustrations.py`.

- **Purpose**: Visual illustrations for Japanese grammar patterns
- **Storage**: Images and metadata (JSON) stored here and referenced in Neo4j
- **Status**: **Active** - Do not remove or move

### `images/lessons/` (Empty)

**✅ REMOVED**: The legacy `images/lessons/cando/` directory has been migrated and removed.

All lesson images are now in `frontend/public/images/lessons/cando/` where Next.js can serve them directly.

## Image Generation Locations

- **Grammar images**: `images/grammar/` (active)
- **Lesson images**: `frontend/public/images/lessons/cando/` (active)

## Migration (Completed)

✅ Migration completed: All legacy lesson images have been migrated from `images/lessons/cando/` to `frontend/public/images/lessons/cando/`.

The migration script (`backend/scripts/migrate_lesson_images.py`) is kept for reference but is no longer needed.
